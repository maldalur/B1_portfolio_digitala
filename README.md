# B1 Portfolio Digitala

Este proyecto contiene una colección de recursos y proyectos relacionados con visión por computadora y aprendizaje profundo. Está organizado en diferentes carpetas que incluyen notebooks, scripts y archivos de apoyo para explorar conceptos clave y realizar prácticas.

## Estructura del Proyecto

- **audioReader.js**: Archivo JavaScript para la lectura de audio.
- **bidalketa1.html - bidalketa6.html**: Archivos HTML que representan diferentes páginas del proyecto.
- **index.html**: Página principal del proyecto.
- **styles.css**: Archivo CSS para los estilos del proyecto.
- **Deep_Computer_Vision_Proiektuak/**: Carpeta que contiene proyectos relacionados con visión por computadora.
  - **01_CNN_Oinarriak/**: Introducción a redes neuronales convolucionales (CNN).
  - **02_Transfer_Learning/**: Ejercicios de Transfer Learning.
  - **03_Objektu_Detekzioa/**: Proyectos de detección de objetos.
  - **04_Aurpegi_Ezagutza/**: Proyectos de reconocimiento facial.
  - **05_Proiektu_Finala/**: Proyecto final utilizando el conjunto de datos CIFAR-10.

## Requisitos

Para trabajar con los notebooks y scripts incluidos en este proyecto, asegúrate de instalar las dependencias necesarias. Puedes encontrarlas en el archivo `requirements.txt` dentro de la carpeta `Deep_Computer_Vision_Proiektuak/`.
